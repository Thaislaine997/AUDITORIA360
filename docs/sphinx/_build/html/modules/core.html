Validadores¶

Validation utilities for AUDITORIA360. Consolidated from services/core/validators.py

src.core.validators.validate_cpf(cpf: str) → bool[código-fonte]¶

Validate Brazilian CPF (Cadastro de Pessoas Físicas) number.

Performs comprehensive validation including format, length, and check digit verification according to Brazilian government standards.

Parâmetros:

cpf – The CPF string to validate. Can contain formatting characters (dots, hyphens) that will be automatically removed.

Retorna:

True if the CPF is valid, False otherwise.

Exemplo

>>> validate_cpf("123.456.789-09")
True
>>> validate_cpf("12345678909")
True
>>> validate_cpf("111.111.111-11")
False
>>> validate_cpf("invalid")
False

Nota

This function validates the mathematical correctness of the CPF but does not verify if it’s actually registered in government databases.

src.core.validators.validate_cnpj(cnpj: str) → bool[código-fonte]¶

Validate Brazilian CNPJ (Cadastro Nacional de Pessoas Jurídicas) number.

Validates CNPJ format and check digits according to Brazilian standards.

Parâmetros:

cnpj – The CNPJ string to validate. Accepts formatted (XX.XXX.XXX/XXXX-XX) or unformatted (XXXXXXXXXXXXXX) strings.

Retorna:

True if CNPJ is valid, False otherwise.

Exemplo

>>> validate_cnpj("11.222.333/0001-81")
True
>>> validate_cnpj("11222333000181")
True
>>> validate_cnpj("00.000.000/0000-00")
False
Levanta:

ValidationError – If input is not a string or is empty.

src.core.validators.validate_phone_br(phone: str) → bool[código-fonte]¶

Validate Brazilian phone number format.

Accepts mobile and landline numbers with or without country code.

Parâmetros:

phone – Phone number string. Can include formatting characters.

Retorna:

True if phone number format is valid, False otherwise.

Exemplo

>>> validate_phone_br("(11) 99999-9999")
True
>>> validate_phone_br("+55 11 99999-9999")
True
>>> validate_phone_br("11999999999")
True
>>> validate_phone_br("(11) 3333-3333")
True

Nota

Validates format only, not if the number is actually active.

src.core.validators.validate_payroll_data(data: dict) → tuple[bool, list[str]][código-fonte]¶

Comprehensive validation for payroll data entries.

Validates employee payroll information according to Brazilian labor laws and internal business rules.

Parâmetros:

data – Dictionary containing payroll information with keys: - employee_id (str): Employee identification - salary (float): Base salary amount - hours_worked (float): Hours worked in the period - benefits (dict): Additional benefits data

Retorna:

  • bool: True if all validations pass, False otherwise

  • list[str]: List of validation error messages (empty if valid)

Tipo de retorno:

Tuple containing

Exemplo

>>> data = {
...     "employee_id": "EMP001",
...     "salary": 2500.00,
...     "hours_worked": 160.0,
...     "benefits": {"health": 150.0}
... }
>>> is_valid, errors = validate_payroll_data(data)
>>> is_valid
True
>>> errors
[]
Levanta:

ValidationError – If data is not a dictionary or required fields are missing.
